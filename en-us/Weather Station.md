### How to Sync Your iOS Calendar with SwitchBot E-Ink Home Dashboard

You can sync personal calendars from Outlook, iCloud, Outlook, and Yahoo to E-Ink Home Dashboard using an ICS file format URL. Once the sync is complete, your personal calendar schedule will be viewable on the E-Ink Home Dashboard display. Please follow the steps below to obtain the Outlook Calendar sharing link and sync it to E-Ink Home Dashboard.

Obtaining the URL of an ICS File Format from Outlook Calendar
1. Launch Outlook Calendar and log in.  
2. Click the gear icon in the upper right corner to go to the settings screen.  
3. Click "Shared Calendars" under "Calendar".  
4. Scroll down, select the calendar you want to sync, set the permissions, and then click "Publish" to generate the link.  
5. Copy the ICS link.

Synchronizing the Shared URL with the E-Ink Home Dashboard
1. Make sure your SwitchBot App version is up to date.
2. Make sure SwitchBot E-Ink Home Dashboard has been added to your App and your device firmware version is up to date.
3. In the SwitchBot app, go to E-Ink Home Dashboard 鈫?Calendar Settings 鈫?Sync, then paste the acquired iCal format private URL and name the calendar.
4. Once the setup is complete, the data will be updated automatically. If it has not been updated, you can manually refresh it by pressing the button on the top of the device.

### Why Does the Screen of SwitchBot E-Ink Home Dashboard Flicker Periodically?

SwitchBot E-Ink Home Dashboard features a monochrome e-ink display. When the partial refresh count for the clock, temperature, and humidity display reaches a specific threshold, a full-screen refresh is required to maintain display quality. Additionally, to ensure weather data accuracy, the E-Ink Home Dashboard automatically connects to the internet every 3 hours to update weather information. During this data refresh, the device will perform a full-screen refresh. The resulting screen flickering is a characteristic of the device and does not affect normal operation. Please feel free to use it.

### How to Set Temperature and Humidity Alert Conditions for SwitchBot E-Ink Home Dashboard

1. Please set alert conditions in the following order.
- Launch SwitchBot App and log in.
- Tap E-Ink Home Dashboard.
- Tap the gear icon in the upper right corner.
- Tap "Alert Conditions".
- Enable and set the alert range for temperature, humidity, absolute humidity, dew point, and VPD as needed.
- Enable "Alert Sound & Light" if you would like the screen to flash or an alarm to sound when the alert setpoint is reached.
